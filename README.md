# Visual-Attention-of-Block-Design-Test-Data

This repository provides supplementary experimental outputs associated with the proposed visual attention assessment framework for the Block Design Test (BDT). The repository contains visual attention recordings, generated graphs, adjacency matrices, and visualization outputs used during the evaluation process.

---

## Overview

The proposed framework analyzes participants' visual attention behavior during the Block Design Test by transforming eye-gaze transitions into graph representations. Each participant's visual exploration pattern is modeled as a directed graph, enabling quantitative comparison with a reference graph using graph matching techniques.

The repository is intended to support reproducibility and provide additional experimental materials that cannot be fully presented within the manuscript.

## Research Context

These materials were generated as part of the study:

**"Visual Attention Assessment Framework for the Block Design Test Using Graph-Based Analysis"**

The framework evaluates participants' visual attention strategies by:

1. Acquiring gaze data using an eye-tracking device.
2. Mapping gaze coordinates into predefined AOIs.
3. Constructing directed graphs from gaze transitions.
4. Comparing participant graphs against a reference graph.
5. Quantifying similarity using graph matching techniques.
